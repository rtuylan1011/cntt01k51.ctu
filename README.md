# cntt01k51.ctu

**1. Giới thiệu**
Đây là trang web giới thiệu và lưu trữ các hoạt động liên quan đến lớp Công nghệ thông tin 01 K51 thuộc trường Công nghệ thông tin và Truyền thông - Đại học Cần Thơ.

**2. Giao diện**

**3. Các tính năng chính**
_3.1. Trang chủ_
    + Thông tin và giới thiệu sơ lược.
    + Các sự kiện nổi bật.
    + Thành tích.
    
_3.2. Giới thiệu_
    + Tổng quát giới thiệu.
    + Ban cán sự các nhiệm kỳ.
    + Ban chấp hành các nhiệm kỳ.
    
_3.3. Hoạt động_
    + Hoạt động nổi bật đang diễn ra.
    + Hoạt động đã hoàn thành và báo cáo thành tích.
    
_3.4. Thành tích_
    + Các thành tích tập thể
    + Các thành tích cá nhân trong phong trào "Sinh viên 5 tốt"...
    
_3.5. Kế hoạch_
    Lưu trữ bản kế hoạch của các sự kiện hoạt động cấp chi đoàn.
    
_3.6. Liên hệ_
    Email chi đoàn Công nghệ thông tin 01 K51
    Facebook chi đoàn Công nghệ thông tin 01 K51

**4. Công nghệ sử dụng**
Frontend: HTML5, CSS3, JavaScript.
Công cụ: Git, GitHub Pages

**5. Hướng dẫn cài đặt và chạy thử**
Bước 1: 
Clone source code về máy: git clone https://github.com/rtuylan1011/cntt01k51.ctu.git
Bước 2: 
Di chuyển vào thư mục dự án: cd cntt01k51.ctu
Bước 3: 
Chạy web
